# 版本管理

bws 提供了完整的浏览器版本管理功能，包括列出版本、查看详情、设置默认版本、卸载版本等。本章详细介绍这些功能的使用方法。

## 列出版本

### 列出所有已安装版本

使用 `ls` 或 `list` 命令列出所有已安装的浏览器版本：

```bash
bws ls
bws ls
```

输出示例：

```
Google Chrome（已安装 3 个，系统 1 个）
  126.0.6478.114 [stable]
  121.0.6167.85
  120.0.6099.109
  125.0.6422.112 [stable] [系统]

Mozilla Firefox（已安装 2 个，系统 0 个）
  121.0 [stable]
  115.0esr [esr]
```

### 列出指定浏览器的版本

可以指定浏览器名称，只列出该浏览器的版本：

```bash
bws ls chrome
bws ls firefox
```

### 使用短别名

支持使用短别名简化输入：

```bash
bws ls gc      # chrome
bws ls ff      # firefox
bws ls cm      # chromium
```

### 列出所有版本（含远程）

使用 `--all` 参数列出所有来源的版本（包括本地和远程）：

```bash
bws ls --all
bws ls --all chrome
```

### 不显示系统浏览器

使用 `--no-system` 参数隐藏系统已安装的浏览器版本：

```bash
bws ls --no-system
```

### 列出版本对比

| 命令 | 说明 |
|------|------|
| `bws ls` | 列出本地已安装版本（含系统） |
| `bws ls --all` | 列出所有版本（本地 + 远程） |
| `bws ls --no-system` | 不显示系统浏览器 |
| `bws ls --remote` / `bws ls -R` | 列出远程可用版本 |

## 部分版本号筛选

bws 支持使用部分版本号进行筛选，只需输入版本号的前缀即可匹配所有以该前缀开头的版本。

### 基本用法

```bash
# 列出所有 chrome 79.x 版本
bws ls chrome@79

# 列出所有 chrome 120.0 版本
bws ls chrome@120.0

# 列出所有 firefox 115 版本
bws ls ff@115
```

### 匹配规则

- 输入 `chrome@79` 会匹配 `79.0.3945.79`、`79.0.3945.130` 等所有 79.x 版本
- 输入 `chrome@120.0` 会匹配 `120.0.6099.109`、`120.0.6099.112` 等版本
- 匹配结果按版本号从高到低排序
- 当需要选择具体版本时（如 run 命令），默认选择最新的匹配版本

### 部分版本号匹配输出示例

运行 `bws r chrome@85` 时的匹配输出：

```
匹配 chrome@85 的版本：
> 85.0.4183.121
  85.0.4183.83
  85.0.4183.10
```

其中 `>` 标记表示当前选中的版本（最新版）。

### 支持的命令

部分版本号筛选在以下命令中都可以使用：

- `bws ls` - 列出版本
- `bws show` - 查看版本详情
- `bws r` - 运行浏览器
- `bws i` - 安装版本
- `bws u` - 设置默认版本
- `bws rm` - 卸载版本
- `bws dl` - 仅下载
- `bws pf` - Profile 管理

## 查看版本详情

使用 `info` 命令查看指定版本的详细信息：

```bash
bws show chrome@120
bws show chrome@120.0.6099.109
bws show gc@system
```

输出内容通常包括：

- 浏览器名称和版本号
- 发布渠道（stable、beta、dev、canary、esr 等）
- 安装路径
- 架构信息
- Profile 路径
- 可执行文件路径
- 安装来源（本地导入、远程下载、系统安装）

## 设置默认版本

使用 `use` 命令设置默认浏览器版本，设置后可以直接用浏览器名运行而无需指定版本。

### 设置默认版本

```bash
bws u chrome@120
bws u ff@121
```

### 使用默认版本

设置默认版本后，可以直接运行：

```bash
bws r chrome
bws r firefox
```

### 查看当前默认版本

```bash
bws cfg get default-browser
```

### 支持部分版本号

`use` 命令也支持部分版本号，会自动选择最新的匹配版本：

```bash
bws u chrome@120     # 自动选择最新的 120.x 版本
```

## 卸载版本

使用 `uninstall` 命令卸载指定的浏览器版本。

### 基本用法

```bash
bws rm chrome@120
bws rm chrome@120.0.6099.109
```

### 支持部分版本号

```bash
bws rm chrome@85     # 卸载最新的 85.x 版本
```

### 注意事项

1. 卸载时会删除浏览器程序文件，但不会删除对应的 Profile 数据
2. 如果需要同时删除 Profile 数据，请使用 `bws pf reset` 命令
3. 系统安装的浏览器无法通过 bws 卸载
4. 卸载前会显示确认提示，确认后才会执行删除
5. 卸载后，残留的 Profile 数据可以通过 `bws pf clean` 清理

### 卸载确认

执行卸载命令时，会显示确认提示：

```
确定要卸载 chrome 120.0.6099.109 吗？(y/N)
```

输入 `y` 确认卸载，输入 `n` 或直接回车取消。

## 架构兼容性

bws 会自动检测系统架构，并过滤不兼容的浏览器版本：

- **x64 系统**：可以运行 x64 和 x86 版本
- **x86 系统**：只能运行 x86 版本
- **arm64 系统**：只能运行 arm64 版本

在列出版本和导入版本时，会自动隐藏不兼容的架构版本，避免误用。
