快速配置

在客户端配置此服务器为远程源：

bws config set source {{.BaseURL}}

配置完成后，即可通过 bws ls -R 查看可用版本并使用 bws install 命令安装。

开始使用

查看可用的浏览器版本并安装运行：

# 查看可用的远程版本
bws ls -R chrome

# 安装指定版本
bws install chrome@120

# 运行浏览器（前台模式）
bws run chrome@120

# 后台运行浏览器
bws run chrome@120 -d
