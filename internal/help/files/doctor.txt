================================================================================
                          bws doctor - 系统健康检查
================================================================================

【命令说明】
  doctor 命令用于检查 bws 运行环境的健康状况，诊断潜在问题。

【命令格式】
  bws dt

【检查内容】

  ┌───────────────────┬──────────────────────────────────────────┐
  │ 检查项             │ 说明                                     │
  ├───────────────────┼──────────────────────────────────────────┤
  │ 目录结构           │ 数据目录、配置文件等必需目录是否存在       │
  │ 配置文件           │ 配置是否能正常加载                         │
  │ 浏览器描述符       │ 支持的浏览器类型是否可用                   │
  │ 已安装版本         │ 已安装的浏览器版本数量                     │
  │ 系统浏览器         │ 是否检测到系统安装的浏览器                 │
  │ 远程源             │ 远程版本信息是否可用                       │
  │ 下载支持           │ 下载管理器是否可用                         │
  └───────────────────┴──────────────────────────────────────────┘

【使用示例】

  bws dt

【输出说明】
  * ✓ 表示该项检查通过
  * ✗ 表示该项存在问题
